2017-08-02



Mickael
I was looking for a free Gallery for the children sportclub homepage. I have try the nanogallery and like we have picture of children it was necessary for us to own the picture by use, so we have used the nano_photos_provider2 too. Everythings works without any xml or Database, you have one config file where you can set all parameters. It allows us to have different pathes for the php form, the script form and the picture folder, so you can integrate it in your own web site structure.
I got a small issue on first installation, I was in contact with the developper Chris. He is very friendly, and has found a solution and published a new version in one day.
Thanks for this product and all your job. With this Gallery, everyone can publish picture in a professional form without beeing an IT specialist.

Recommand nanogallery to other people: Yes
If you could change one thing what would it be: nothings to change
